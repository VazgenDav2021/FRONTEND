1. UseEffect - это хук который срабатывает во время изменний зависимостей
2. UseEffect как параметры принимает колбэк функцию и массив зависмостей.
3. Колбек функция это функция которая вызывается столько раз сколько менятся зависимости указанные массивы
4. Если в массив зависимостей не указаать то функция которая написанна в useEffect сработает бесконченое кличество раз
5. Если в массиве завимостей нет то функция которая написанн в useEddect сработает один раз
6. Если в массиве завимостей указать значение то функция будет вызыватся столько раз сколько раз будут обновлятся значение в массиве заивимостей
